# TodoApp Client

A React 18 frontend application for the TodoApp backend, built with Fluent UI 9.

## Prerequisites

- Node.js v26 or later
- TodoApp backend running on http://localhost:5025

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The application will be available at http://localhost:3000

## Features

- View all todos in a sortable data grid
- Display todo name, details, created date, and due date
- Modern UI with Fluent UI 9 components

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Tech Stack

- React 18
- TypeScript
- Fluent UI 9
- Vite
